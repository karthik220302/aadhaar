Working Flow


  1. First user needs to click change address button in order to change or update the address
  2. Now we get the Aadhaar number of the user and Then the user needs to enter the captcha
  3. After entering this , an otp is sent to the users mobile phone ensuring that the process is completely secured
  4. After entering the otp correctly the user is directed to the change of address page where he needs to enter the houseowner/landlords MOBILE number
  5. After this the user is directed to the status page where he gets to know about the status of the application to change/update address
  6. In the mean while houseowner/landlord gets a notification containing the details about the user who have requested for change of address and a link to check the request
  7. If the houseowner/landlord approves the request then he is directed to a page where we request his aadhaar number and after entering this
  8. He is directed to the final process where he needs to set a passcode and enter the otp sent to his mobile after this process houseowners task is completed
  9. Now after the houseowner approves the user request he/she gets the status of the application and then user is directed to a page where user gets the address of the houseowner/landlord and here he can make any small changes he wish to do
  10. After this the address entered by the user is let inside the address checking algorithm and if there is no big change then the user request is accepted and then his/her address is updated within a blink
